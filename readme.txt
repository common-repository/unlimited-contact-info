== Description ==

Unlimited Contact Info is a plugin that allows you to configure the fields of contact for member profiles.

You can disable the default contact information (AIM, Yahoo IM and Jabber / Google Talk.

In addition to this, the plugin can add an unlimited number of new areas of contact.

After saving changes the new fields, the code inserted in the template file to display the field value is indicated below each field.

Translation: English, French, Italian, Spanish

== Installation ==

1. Upload the complete `unlimited-contact-info` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Unlimited Contact Info' under the 'Settings' tab and configure the plugin


== Frequently Asked Questions ==

= How many contact fields can we add ? =

Unlimited! This is the advantage of this plugin. You can add as many new fields as you want.

= How to display the contact fields on the member's profile page ? =

In the setup menu of the plugin, the code inserted in the template file is shown below each field after saving changes.
